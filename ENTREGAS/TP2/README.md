# Machine Learning

Modelo de Machine Learning sobre la predicci√≥n de si un hongo es comestible o no. El objetivo es explicar y predecir la variable class, que vale p, si el hongo es venenoso (poisonous), o e, si el hongo es comestible (edible).

El proyecto se divide en 4 partes.
